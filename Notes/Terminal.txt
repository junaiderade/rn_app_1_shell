游리 Install cocoa pods prior to building for ios
sudo gem install cocoapods
游리 Install react Navigation
npm install @react-navigation/native @react-navigation/native-stack
游리 Install peer dependencies for Expo for react navigation
expo install react-native-screens react-native-safe-area-context
游리 Check node version
node -v
游리 run expo and clear your expo cache
expo start -c
游리 start the app frominside the app folder (which contains package.json)
npm start OR expo start
游리 see what branch you are on
git branch
游리 how to clone a project from github
- make a folder and do git init
- then do git clone <github url>
- do npm install to install dependencies
游리 install yarn
npm i -g yarn
游리 how to start expo on PC when there is an issue connecting
expo start --tunnel
游리 how to pull latest code from github while taking local changes into consideration
git fetch origin
git status (then you will see something about how behind you are or what conflicts you have)
git pull (once conflicts are resolved)
游리 create a new branch and switch to it
git checkout -b <new branch name>
